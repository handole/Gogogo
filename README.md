# Latihan Golang Dasar

Dokumen ini berisi latihan dasar pemrograman menggunakan bahasa Go (Golang).

## Daftar Latihan

1. **Hello World**
2. **Variabel dan Tipe Data**
3. **Pengkondisian (if, switch)**
4. **Perulangan (for)**
5. **Fungsi**
6. **Array dan Slice**
7. **Struct**
8. **Map**
9. **Pointer**
10. **Package dan Modul**

## Contoh Hello World

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
